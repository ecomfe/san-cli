# 查看webpack内置信息

## 查看 webpack rule

### 查看所有内置 webpack rules list

```bash
san inspect --rules
```

```js
[
  'san',
  'js',
  'ejs',
  'html',
  'svg',
  'img',
  'media',
  'fonts',
  'css',
  'postcss',
  'less'
]
```

### 查看具体内置 webpack rule 信息

```bash
san inspect --rule postcss
```

```bash
san inspect --plugins
```
